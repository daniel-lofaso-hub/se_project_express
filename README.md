# WTWR (What to Wear?): Back End

This project is an introduction into using Express. In the project, we use ESLint, Prettier (with an exception from Airbnb's Javascript Style Guide for underscores), and Mongoose. We connect to a MongoDB server, creating models, schemas, routes, and controllers from scratch. We also implemented the appropiate error handling throughout the project, as well as using JSONWebToken and bcrypt to create, transfer, and store user data securely.
